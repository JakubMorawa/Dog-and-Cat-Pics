Welcome to Dog and Cat Pics!

Press the Random Cat and Random Dog buttons to get your own random adorable picture including some infomation of the pet if available.

How was this website made?

Using the Cat / Dog API, you are able to grab different images of pets and even their different attributes. The backend is operated by Python and the Flask microframework, which allows me to call the API and grab different elements from the data and display it in the website by the Jiinja web template engine.


Have fun looking at all these pets!
